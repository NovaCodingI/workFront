정규표현식(Regular Expresision)

정규 표현식(Regular Expresision)이란,
특정 규칙을 가진 문자열을 검색하거나 치환할 때 사용하는 형식 전화번호, 이메일, 파일의 확장자의 형식을 확인 할때 사용

1. 정규 표현식 객체 생성

정규표현식 리터럴 RegExp생성자 함수를 이용

2. 플래그 문자

정규표현식의 검색 방식을 설정 순서와 상관없이 여러개를 지정 할 수 있다

  • i : 대소문자를 가리지 않고 비교
  • g : 전역으로 비교를 수행
  • m : 여러줄의 비교를 수행

3. 메타문자

특정 메타 문자를 통해 정규식으로 특정 패턴을 제시 할 수 있다
